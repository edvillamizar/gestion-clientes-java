# Gestión de Clientes — Java

Aplicación de escritorio para la gestión de clientes desarrollada en Java.
Permite registrar, consultar, actualizar y eliminar clientes mediante una interfaz sencilla.

## 🛠️ Tecnologías

- Java SE
- Programación orientada a objetos (POO)
- Operaciones CRUD
- Consola / interfaz básica

## ⚙️ Funcionalidades

- Crear nuevo cliente
- Listar todos los clientes
- Actualizar datos de un cliente
- Eliminar cliente por ID

## ▶️ Cómo ejecutarlo

1. Clona el repositorio:
```bash
   git clone https://github.com/edvillamizar/gestion-clientes-java.git
```
2. Abre el proyecto en tu IDE (Eclipse, IntelliJ o VS Code con extensión Java)
3. Ejecuta la clase principal `Main.java`
